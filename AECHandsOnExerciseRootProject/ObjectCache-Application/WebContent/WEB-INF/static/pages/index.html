Sapient ACE-Assignment : Object Cache

ObjectCache-Application is a Object Cache use-case implementation. Application uses Object Cache module for the implementation.
"Object Cache" module is a LRU bases cache implementation and uses underlying ConcurrentHashMap to store cached Objects

Cache Eviction Policy

1: LRU - Least recently Used

"Object Cache" removes Least Recently Used objects when storing new object and size is overflow (hence LRU)


2: Time bound

Object to be fetch from cache,

1:if expired(exceeds timeToLive in cache), the Object will be removed from cache and a fresh call will be triggered to Object-Finder to fetch fresh Object from repository.

2:if calls happen within expired(exceeds timeToLive in cache) time, the Object's timeToLive will be Renewed in cache. No call will be triggered to Object-Finder.
